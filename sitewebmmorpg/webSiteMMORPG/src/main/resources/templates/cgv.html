Introduction

Ce document détaille les spécifications du projet MMORPG-VORTX, couvrant les aspects du jeu, du site web, des bases de données et des serveurs. Il vise à garantir une conformité stricte avec les exigences du client et des développeurs.

Serveur

  • Le serveur est basé sur Ubuntu et utilise Java.
  • Configuration minimale : 8 Go de RAM, processeur équivalent à un i3, connexion filaire.
  • Les commandes incluent help, save, et forceExit.

Site Web

Le site web permet aux utilisateurs de télécharger le jeu, consulter des articles, participer à des discussions et acheter des loot boxes.

  • Fonctionnalités principales : login, register, blog, wiki, forum.
  • Doit être accessible via tout navigateur moderne.

Jeu

  • Compatible avec Windows, macOS, et Linux.
  • Supporte l’exploration de cartes, les interactions entre joueurs et PNJ, et les quêtes.
  • Basé sur LibGDX pour la gestion des collisions et de la physique.

Base de Données

La base de données est hébergée sur un serveur MySQL. Elle gère des données en temps réel pour le jeu et le site web.

  • Configuration minimale : Compatible MySQL, sécurisé avec accès restreint.
  • Support des mises à jour fréquentes et journaux d'accès pour la sécurité.

Contraintes

  • Budget limité à 0€ pour le serveur et le jeu.
  • Doit être optimisé pour fonctionner sur des machines peu puissantes.
  • Respect strict des normes de sécurité des données (chiffrement, journalisation).
